# To-Do List Web App

A simple To-Do List web application using HTML, CSS, and JavaScript.

## How to Use
Open `index.html` in your browser, or see the deployed GitHub Pages link.

## Features
- Add, complete, and delete tasks
- Saves tasks in your browser's local storage

## View Live
After deployment, your GitHub Pages site will likely be at:

```
https://onkarlonsane-stack.github.io/first/
```

## Deploying/Updating GitHub Pages
If not already enabled:
1. Go to your repository's **Settings** > **Pages**.
2. Under **Source**, select the `main` branch and `/ (root)` folder.
3. Save. Wait a few moments and visit the link above.
